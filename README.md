# Muhammad Nishmal P - Developer Portfolio

A high-performance, minimalist portfolio website built with **Next.js 14+**, **Tailwind CSS**, and **TypeScript**. Designed for speed, accessibility, and professional impact.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (v4 configuration)
- **Icons:** Lucide React
- **Font:** Inter (Google Fonts)
- **Deployment:** Vercel

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This project is optimized for Vercel.

1. Push to GitHub.
2. Import project in Vercel.
3. Deploy (Zero configuration required).

## Project Structure
- `app/`: Source code and pages.
- `components/`: Reusable UI components and sections.
- `data/`: Structured content for projects, skills, and experience.
- `lib/`: Utility functions.
